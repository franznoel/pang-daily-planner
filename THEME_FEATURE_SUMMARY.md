# Theme Selection Feature - User Guide

## Overview

The Pang Daily Planner now allows users to personalize their experience with 9 unique themes representing different generations and identities. Each theme has been carefully designed with catchy names and distinctive color palettes.

## Accessing Themes

### Step 1: Open the User Menu
- Look for your profile avatar in the top-right corner of the app's navigation bar
- Click on the avatar to open the user menu

### Step 2: Select Theme Option
- In the dropdown menu, you'll see a new "Select Theme" option (with a palette icon ðŸŽ¨)
- This option appears at the top of the menu, above "Shared With Me"

### Step 3: Choose Your Theme
- A dialog will open showing all available themes
- Themes are organized into two categories:
  - **Generation Themes**: Inspired by different generational aesthetics
  - **Identity Themes**: Celebrating different communities and identities

### Step 4: Apply Theme
- Click on any theme to apply it immediately
- The selected theme will show a checkmark icon âœ“
- Close the dialog - your theme is now active!

### Step 5: Persistence
- Your theme choice is automatically saved
- The theme will remain active even after you close and reopen the app

## Technical Features

### What Changes with Themes?
- **Colors**: Primary and secondary colors throughout the app
- **Backgrounds**: Page and component backgrounds
- **Typography**: Font families that match the theme's personality
- **Overall Vibe**: Each theme creates a unique atmosphere

### Data Persistence
- Themes are saved locally in your browser
- Your preference syncs across all tabs in the same browser
- Each browser maintains its own theme preference

### Compatibility
- Works with all modern browsers (Chrome, Firefox, Safari, Edge)
- Fully responsive - themes work on desktop, tablet, and mobile
- No performance impact - themes load instantly

## Frequently Asked Questions

**Q: How do I reset to the default theme?**
A: Select "Digital Dawn" from the theme menu. This is the default Millennials theme.

**Q: Can I create my own custom theme?**
A: Currently, you can choose from the 9 pre-designed themes. Custom theme creation may be added in the future.

**Q: Will my theme choice be lost if I clear my browser data?**
A: Yes, clearing localStorage will reset your theme to the default. Simply select your preferred theme again.

**Q: Do themes sync across different devices?**
A: Currently, themes are stored locally per browser. Cross-device syncing may be added in future updates.

**Q: Can I preview themes before selecting them?**
A: The theme applies immediately when selected. You can easily switch between themes to find your favorite.

**Q: Are more themes planned for the future?**
A: The system is designed to be extensible, so additional themes can be added based on user feedback.
